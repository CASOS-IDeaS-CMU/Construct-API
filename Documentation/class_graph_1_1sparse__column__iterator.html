Detailed Description

template<typename T>
class Graph< T >::sparse_column_iterator

Iterator that iterates over all elements except elements specified by get_sparse_row at a constant row index.

In a sparse representation, this iterator will skip over elements that equal the submitted variable skip_data in get_sparse_row. If skip_data equals the default value, this will increment over each missing element. Because this sparse iterator, given appropriate skip_data and default value, can iterate over elements that are missing, a hard examine can still create a new element at the iterator's position and then point to that element. The soft examine will point to the default value if the element is missing. In a dense representation, the iterator will continue to increment until an element not equal to the skip_data is found or the end of the row is reached.

While an iterator can be created using the default constructor, they can only be initialized by a Graph class. An uninitialized iterator should not be dereferenced or incremented. Likewise an iterator equal to the end of a row should not be dereferenced or incremented.

template<typename T >
void Graph< T >::sparse_column_iterator::clear ( void  )

Resets the element currently being pointed to.

If the column density is dense, the element is set to the default value. If the density is sparse, the element is removed.

Example

int main()
{
// Graph<int>* mygraph has been initialized with the following data
// 0 1 4 2
// 1 0 0 0
// 0 0 2 3
// 4 5 3 6
// each zero reflects an element that is not present with the default value for mygraph being zero.
std::cout << "row 0:" << std::endl;
Graph<int>::sparse_column_iterator it, end;
for (mygraph.get_sparse_row(0, it, end, 0); it != end; ++it) {
if (it.col() == 3) it.clear();
std::cout << " " << *it;
}
std::cout << std::endl;
}
