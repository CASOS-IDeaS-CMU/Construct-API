Detailed Description

template<typename T>
class Graph< T >::const_sparse_col_iterator

A constant iterator that iterates over each element whose value does not equal the skip value.

If no value is found to not equal the skip value when the iterator is incremented, the iterator is advanced to the end iterator. This iterator is identical to Graph::sparse_col_iterator in function, however this iterator can't be submitted as a parameter to at or clear.
