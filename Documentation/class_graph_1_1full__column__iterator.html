Detailed Description

template<typename T>
class Graph< T >::full_column_iterator

Iterator that iterates over all elements at a constant row index.

In a sparse representation, this iterator will increment through every index in the target dimension. If an element is missing, a soft examine will point to the default value and a hard examine will create a new element at the iterator's position and will then point to that element. In a dense representation, the iterator increments through all elements as expected.

While an iterator can be created using the default constructor, they can only be initialized by a Graph class. An uninitialized iterator should not be dereferenced or incremented. Likewise an iterator equal to the end of a row should not be dereferenced or incremented.

template<typename T >
void Graph< T >::full_column_iterator::clear ( void  )

Resets the element currently being pointed to.

If the column density is dense, the element is set to the default value. If the density is sparse, the element is removed.

Example

int main()
{
// Graph<int>* mygraph has been initialized with the following data
// 0 1 4 2
// 1 0 0 0
// 0 0 2 3
// 4 5 3 6
// each zero reflects an element that is not present with the default value for mygraph being zero.
std::cout << "row 0:" << std::endl;
Graph<int>::full_column_iterator it, end;
for (mygraph.get_full_row(0, it, end); it != end; ++it) {
if (it.col() == 3) it.clear();

}
std::cout << std::endl;
