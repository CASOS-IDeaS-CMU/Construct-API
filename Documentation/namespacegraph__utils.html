void CONSTRUCT_LIB graph_utils::init_align ( std::vector< Graph_iterator * > &  it_list)

Aligns the iterators if they are not already.

Does nothing if all Graph_iterator::index are equal. Otherwise it_align is used to align the iterators.

Parameters
it_listA vector of iterator pointers this function will align.

Example

int main()
{
Graph<int>* mygraph;
// Graph<int>* mygraph has been initialized with the following data
// 0 1 4 2
// 1 0 0 0
// 0 0 2 3
// 4 5 3 6
// the default value for mygraph being zero.
// each element with a zero does not have a data entry
auto iit = mygraph->full_row_begin(3);
auto jit = mygraph->sparse_row_begin(2, 0);
std::vector<Graph_iterator*> it_list = { &iit, &jit };
for(graph_utils::init_align(it_list); iit != mygraph->row_end(3); graph_utils::it_align(it_list)) {
std::cout << "Agent 3 has a value of " << *iit << " and agent 2 has a value of " << *jit << std::endl;
}
}

void CONSTRUCT_LIB graph_utils::it_align ( std::vector< Graph_iterator * > &  it_list)

Finds the next index that aligns iterators.

Each iterator is advanced until each Graph_iterator::index are all equal. If an iterator's index ever equals Graph_iterator::max, all other iterators are advanced until they are set to their maximum index.

void CONSTRUCT_LIB graph_utils::it_align_before_first ( std::vector< Graph_iterator * > &  it_list)

Finds the next index that aligns iterators except the first iterator.

The first iterator is incremented if its Graph_iterator::index is less than any of the other iterators in the vector. If the first iterator index is ever equal to any other iterator, both iterators are incremented.

it_listA vector of iterator pointers this function will align.
