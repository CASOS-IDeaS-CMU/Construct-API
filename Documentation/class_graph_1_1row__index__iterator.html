Detailed Description

template<typename T>
class Graph< T >::row_index_iterator

Iterates through all real row indexes of a graph

In the case the source dimension is dense, the iterator will increment from 0 to the source nodeset size. If the dimension is sparse, the iterator will only go to indexes in which that row actually exists. This does not take any information on what the default value for the graph is.

template<typename T >
Graph< T >::row_index_iterator::row_index_iterator ( )
inline

Default constructor.

This constructor creates an uninitialized iterator. This iterator is initialized by a Graph using get_rows. As with all uninitialized iterators, this iterator should not be dereferenced or incremented when uninitialized. Dereferencing this initialized iterator will yield the index of the row this iterator is pointing to. Incrementing will point the iterator to the next real row in the corresponding Graph.

Example

int main()
{
// Graph<int>* mygraph has been initialized with the following data
// 0 1 4 2
// 1 0 0 0
// 0 0 0 0
// 4 5 3 6
// each zero reflects an element that is not present with the default value for mygraph being zero.
// the source dimension has been made sparse and thus row 2 does not actually exist.
std::cout << "Rows that are real:";
Graph<int>::row_index_iterator row, rend;
for (mygraph.get_rows(row, rend); row != rend; ++row) {
std::cout << " " << *row;
}
std::cout << std::endl;
}
