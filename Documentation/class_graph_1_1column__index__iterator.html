Detailed Description

template<typename T>
class Graph< T >::column_index_iterator

Iterates through all real column indexes of a graph

In the case the target dimension is dense, the iterator will increment from 0 to the target nodeset size. If the dimension is sparse, the iterator will only go to indexes in which that column actually exists. This does not take any information on what the default value for the graph is.

template<typename T >
Graph< T >::column_index_iterator::column_index_iterator ( )
inline

Default constructor.

This constructor creates an uninitialized iterator. This iterator is initialized by a Graph using get_columns. As with all uninitialized iterators, this iterator should not be dereferenced or incremented when uninitialized. Dereferencing this initialized iterator will yield the index of the column this iterator is pointing to. Incrementing will point the iterator to the next real column in the corresponding Graph.

Example

int main()
{
// Graph<int>* mygraph has been initialized with the following data
// 0 0 4 2
// 1 0 0 0
// 0 0 2 3
// 4 0 3 6
// each zero reflects an element that is not present with the default value for mygraph being zero.
// the target dimension has been made sparse and thus column 1 does not actually exist.
std::cout << "Columns that are real:";
Graph<int>::column_index_iterator col, cend;
for (mygraph.get_columns(col, cend); col != cend; ++col) {
std::cout << " " << *col;
}
std::cout << std::endl;
}
